
# MovieBrowser

> **주요 기능**
> 
네이버 영화 검색 API 이용하여 영화 검색을 한다.  
영화를 클릭하면 웹 브라우저가 열린다.  
검색어를 저장하여 최근 검색 목록을 볼 수 있다.  
최근 검색 목록에서도 검색 할 수 있다.  


  > **사용 기술**
  > 
Room  
Retrofit  
RecyclerView  
WebView  
ViewBinding
